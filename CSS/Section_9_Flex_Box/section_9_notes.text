1. display: flex
----------------
    The display: flex property is used to create a flex container ‚Äî enabling flexible and responsive layouts.

    Ex:
        .container {
            display: flex;
            gap: 10px
        }

        display: flex; Activates Flexbox layout on the .container.
        gap: 10px: Adds a 10px spacing between the flex items inside the container.
        All child elements inside .container will align horizontally (row by default) with 10px space between them.

        Default Flex Direction:
            flex-direction: row (i.e., left to right)

        ‚úÖ Flexbox: Row and Column Layouts
        -----------------------------------
            ‚úÖ Flex as a Row (default)
            Items are placed horizontally (left to right).

            CSS:
            .container-row {
            display: flex;
            flex-direction: row; /* default */
            gap: 10px;
            }

            HTML:
            <div class="container-row">
            <div>Item 1</div>
            <div>Item 2</div>
            <div>Item 3</div>
            </div>

            -----------------------------------
            ‚úÖ Flex as a Column
            Items are placed vertically (top to bottom).

            CSS:
            .container-column {
            display: flex;
            flex-direction: column;
            gap: 10px;
            }

            HTML:
            <div class="container-column">
            <div>Item A</div>
            <div>Item B</div>
            <div>Item C</div>
            </div>

            -----------------------------------
            üîÑ Responsive Example: Switch Between Row and Column

            CSS:
            .responsive-container {
            display: flex;
            flex-direction: row;        /* row on big screens */
            }

            @media (max-width: 600px) {
            .responsive-container {
                flex-direction: column;   /* column on small screens */
            }
            }

            -----------------------------------
            üîÅ Summary Table

            flex-direction Value | Layout            | Use Case
            ---------------------|-------------------|-------------------------------
            row (default)        | Horizontal (L ‚Üí R) | Nav bars, image galleries
            column               | Vertical (T ‚Üí B)   | Forms, menus, stacked content
            row-reverse          | Horizontal (R ‚Üí L) | Right-to-left layouts
            column-reverse       | Vertical (B ‚Üí T)   | Reverse-stacked elements
